习题及答案

一、选择题

  1. 下面哪个方法与public int max(int x,int y)不是重载方法 ( B )

    A.public double max(double x,double y)

    B.public int max(int n,int k)

    C.public int max(int x,int y, int z)

    D.public double max(double n,double k)

  2. 为了区分类中重载的同名的不同方法，要求 ( B )

    A. 参数列表不同

    B. 调用时用类名或对象名做前缀，比如Login.login()，使用了Login对象名做前缀

    C. 参数名不同

    D. 返回值类型不同

    正确答案：A

    错因：初读B更具有说服力，但重载的方法本就是在同一个类中，使用类名或对象名做前缀也无法区分。对重载的定义尚且模糊。重载要求方法名称相同，参数类型或参数的个数不同。

  3. 设有下面两个类的定义，则类Person和类Student的关系是？( B )

    A.包含关系

    B.继承关系

    C.关联关系

    D.上述类定义有语法错误

  4. 下列叙述中，正确的是 ( C )

    A.子类继承父类的所有属性和方法

    B.子类可以继承父类的私有的属性和方法

    C.子类可以继承父类的公有的属性和方法

    D.创建子类对象时，父类的构造方法都要被执行

  5. 现有两个类A、B，以下描述中表示B继承自A的是 ( D )

    A.class A extends B

    B.class B implements A

    C.class A implements B

    D.class B extends A

  6. 下列叙述中错误的是 ( A )

    A.一个子类可有多个父类

    B.父类派生出子类

    C.子类继承父类

    D.子类只能有一个父类

  7. 下列针对User类中的int类型的私有属性age的操作，正确的是 ( C )

    A. User.age=18; int a=User.getAge();

    B. User.age=18; int a=User.age;

    C. User.setAge(18); int a=User.getAge();

    D. User.setAge(18); System.out.println(User.getAge(18));

  8. 表示数据或方法可以只能被定义它的类访问的是 ( C )

    A.public

    B.默认的

    C.private

    D.protected

  9. 表示数据或方法只能被其它包中的子类访问的是 ( D )

    A.public

    B.默认的

    C.private

    D.protected

  10. 当你编译运行下列程序代码，会得到什么结果 ( C )

    A.这段代码不能通过编译（如果选这个选项，记得去编译器里面看看怎么才能跑起来）

    B.输出200

    C.输出100和200

    D.输出100

    正确答案：A

    解析：

    ​ 若Base和Pri是两个单独的类（外部类），则Base的访问权限不能是private。外部类的访问权限只能有public（公开，任何位置都能被访问到），默认（default，同包访问），故编译不能通过；

    ​ 若两者是在同一个类中（内部类），则Pri中不能有静态属性，这是因为Pri不是一个static类，而static的属性或方法都是在类加载时一直存在于内存中的，而Pri不会被主动加载，故编译不能通过。
