状态修饰符

  1. static（静态）

    • 可以修饰成员变量，成员方法，不能修饰类，不能修饰构造方法
    • 被修饰的变量，方法可以通过类名进行调用

  2. final（最终）

    • 可以修饰类,成员变量,成员方法,不能修饰构造方法
    • 修饰的类不能被继承，被修饰的变量是常量，被修饰的方法不能被继承

抽象修饰符

  • abstract（抽象)

    • 不能修饰构造方法,不能修饰成员变量,可以修饰类,成员方法
    • 被修饰的类不能被实例化，只能被继承，被修饰的成员方法在子类中必须被重写
