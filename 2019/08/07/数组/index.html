学习笔记——Java数组
发表于 2019-08-07| 更新于 2019-08-09|Java
字数总计: 684|阅读时长: 2 分钟|阅读量:

定义

  1. 是一种容器（放东西的东西），特点是：
    1. 其中所有的元素具有相同的数据类型
    2. 一旦创建，不能改变大小
    3. 数组是一种数据结构，能记录同一种类型的多个数据
      1. 数组中的每个数据叫做元素
      2. 所有的元素具有相同的数据类型
  2. 定义数组变量
    1. <类型>[] <名字>=new <类型> [元素个数]；
    2. 示例：int[] numbers[]=new int [100];
    3. 注意：
      1. 元素个数必须是整数
      2. 元素个数必须给出
      3. 元素个数可以是变量

数组的元素

  1. 每个元素都是同种类型的变量
  2. 索引或下标是从0开始的
    1. 最小的下标是0，最大的下标是数组的元素个数-1
    2. 编译器不会检查下标是否有效，都是如果运行时出现了无效的下标可能会导致程序终止
  3. 每个数组固有一个内部成员length，代表数组的元素的数量

数组初始化

  1. new创建的数组会得到默认的0值

  2. 直接赋值，无需给出数组大小，数组的大小通过length得知

    • 示例：int[] scores={87,98,69,54,65};

  3. 数组变量赋值

    • 运行结果为16

    • 解释：

      1. 普通变量是数据的拥有者，而数组变量是数据组的管理者，故数组变量没有数据
      2. a赋值给b，则a,b管理同一个数组

    • 总结

      1. 数组变量是数组的管理者而非数组本身

      2. 数组必须创建出来然后交给数组变量来管理（数组与数组变量不同）

      3. 数组变量之间的赋值是管理权限的赋予

      4. 数组变量之间的比较是判断是否管理同一个数组

遍历数组

  • for_each循环

    • 格式： for(<类型><变量>:<数组>){ ···}
    • 含义：将数组中的数据循环赋给该数据类型的变量

    • data数组中的元素用循环赋给int类型的k

    • 适合遍历一个数组得到数组中的一个数据，进行比较赋值

    • 不适合需要获得数据的位置的情况

    • 注意：通过for-each循环不能修改数组的值，只能得到数组的值

      • 输出结果为data原数据
