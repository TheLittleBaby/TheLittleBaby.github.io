自动转换

  • 程序在执行过程中“悄然”进行的转换，不需要用户提前声明，一般是从位数低的类型向位数高的类型转换（低精度->高精度 转换）
  • 不同类型数据间的优先关系：
    • 低———————————————>高
      byte,short,char-> int -> long -> float -> double

强制转换

  • 必须在代码中声明，转换顺序不受限制（高精度 -> 低精度转换），注意存在精度损失

  • 高———————————————>低
    double->float -> long -> int -> byte,short,char
