字符串变量

  1. 定义

    1. 字符串：用双引号括起来的0个或多个字符就是一个字符串字面量

      • 示例
        • “hello” “1” “ ”

    2. 字符串变量

      1. 示例：String s;

        1. String是一个类（包裹类型），String的变量是对象的管理者而非所有者
        2. 类比数组变量是数组的管理者而非所有者

      2. 格式：new=创建

        • 创建了一个String的对象
        • 用“a string”初始化这个对象
        • 创建管理这个对象的变量s
        • 让s管理这个对象

  2. 初始化字符串变量

  3. 字符串连接

    1. 用加号（+）可以连接两个字符串
      • 示例
        • “hello”+”world”——>”helloworld”
    2. 当加号的一边是字符串而另一边不是时，会将另一边表达为字符串然后做连接
      • 示例
        • “I’m “+18——>”I’m 18”
        • 1+2+”age”——>”3age”
        • “age”+1+2——>”age12”

  4. 输入字符串

    1. in.next(); 读入一个单词，单词的标志是空格
      • 空格包括 空格、tab和换行
    2. in.nextLine(); 读入一整行

字符串操作

  1. 定义

    1. 字符串是对象，对它所有操作都是通过”.”这个运算符来进行的
      • 字符串.操作
    2. 它通过对.左边的这个字符串做右边的操作
    3. 这里的字符串可以是变量也可以是常量

  2. Strings大小的比较

    • 若s1<s2，结果为负；若s1=s2,结果为0；若s1>s2，结果为正
    • compareToIgnoreCase可以不区分大小写地来比较大小

  3. 获得String的长度

    • 用length函数

      • 运行结果 7 3 0

  4. 访问String里的字符

    1. s.charAt(index)
      1. 返回index上的单个字符
      2. index的范围是0到length()-1
      3. 第一个字符的index是0，和数组一样
    2. 不能用for-each循环来遍历字符串

  5. 得到子串

    1. s.substring(n)
      • 得到从n号位置到末尾的全部内容
    2. s.substring(b,e)
      • 得到从b号位置到e号位置（不加上e号位置上的字符)之前的内容

  6. 寻找字符

    1. s.indexOf(c)
      • 得到c字符所在的位置，-1表示不存在
    2. s.indexOf(c,n)
      • 从n号位置开始寻找c字符
    3. s.indexOf(t)
      • 找到字符串t所在的位置
    4. 从右边开始找
      1. s.lastIndexOf(c)
      2. s.lastIndexOf(c,n)
      3. s.lastIndexOf(t)

  7. 其他String操作

    1. 字符串是否以特定的字符串(注意使用双引号）开始或结束
      • s.startsWith()
      • s.endWith()
    2. 删除字符串两端空格
      • s.trim()
    3. 将字符串中的所有c1都换成c2(注意c1,c2只能是单个字符)
      • s.replace(c1,c2)
    4. 将字符串中所有的字母都变成大写或小写的字母
      • s.toLowerCase()
      • s.toUpperCase()

  8. 不可变的String

    • 所有的字符串都是不可变的，对他们的初衷的结果都是制造新的字符串出来
    • 7点以内的操作都是不改变原有的字符串
