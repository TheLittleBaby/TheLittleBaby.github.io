JSP语法

类型 格式 功能 举例 注意
jsp代码块 <% java代码%> 包裹java语句，以区分HTML <% while (x<=10) { sum += x; x++;}%> 可以包含任意合法的java语句
jsp表达式 <%=表达式%> 表达式执行后返回String类型的结果值，并将结果值输出到浏览器上 <%=new Date()%> 1."<%="是完整的标记，中间不能有空格。 2.不能用分号作为表达式的结束符，比如错误写法<%=new Date() ；%>3.表达式元素包含任何在java语言中有效的表达式
jsp声明 <%! 声明变量、方法和类 %> 其中声明的变量，方法和类是jsp页面的成员变量 声明变量：<%! int sum=0,x=1;%> 声明方法：<%! long fact(int y){ if(y==0){ return 1; } else { return y*y; }} %> 在代码块中定义的变量是局部变量，在声明中定义的变量是页面的全局变量
注释 <%--注释内容--%> 在jsp程序中，当在发布网页是完全被忽略，不以HTML格式发给客户 <%-- 不要怕难 --%> 也可使用 1.HTML注释 <!--HTML注释--> 2. java 注释 <%//java注释%>或<% / * **java注释*** / %>
指令 <%@ jsp指令%> 规定该页面所使用的字符编码、使用的工具jar包等信息 <%@ page contentType="text/html;charset=UTF-8" language="java" %> 放在整个页面的开始。一个页面利用使用多个< %@page%>指令，分别描述不同的属性

jsp的page指令元素

  1. page指令用来定义jsp页面中的全局属性，描述与页面相关的一些信息。
  2. <%@ page%>区分大小写。
属性 说明 设置值示例
import 由于导入java包或java类 <%@page import=”Java.util.Date”%>
pageEncoding 指定页面所用编码，默认与contentType值相同 UTF-8
isELIgnord 设置是否忽略正则表达式 true 或 false

  1. include指令（jsp指令元素）、include动作（jsp动作元素）、< jsp:forward >动作、< jsp:param>动作

    类型 格式 功能 注意
    include指令（文件加载指令） < %@ include file=”filename”%> 该指令标签作用是在该标签的位置处，静态插入一个文件 可以将其他的文件插入jsp网页。静态插入是指用被插入的文件内容代替该指令标签，与当前jsp文件合并成新的jsp页面。实现jsp页面的模块化
    include动作 <jsp:include page=”文件的名字”/> 当前jsp页面动态包含一个文件，即将当前jsp页面、被包含的文件各自独立编译为字节码文件。 静态包含不能传递参数，但动态包含可以在两文件之间传递参数
    < jsp:forward >动作 <jsp:forward page=”文件的名字”/> 停止当前页面的执行，转向另一个HTML或jsp页面 浏览器的地址栏中地址不会发生任何变化
    < jsp:param>动作 (1)<jsp:include page=”文件名字”> <jsp:param name=”变量名字1” value=”变量值1”/> ··· < /jsp:include>或 (2)<jsp:forward page=”文件名字”> <jsp:param name=”变量名字1” value=”变量值1”/> ··· < /jsp:forward> (1)将param标签中的变量值传递到动态加载的文件 （2）将将param标签中的变量值传递到要跳转到的文件 略
