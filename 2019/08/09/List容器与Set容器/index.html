学习笔记——List容器与Set容器
发表于 2019-08-09| 更新于 2019-08-17|Java
字数总计: 807|阅读时长: 2 分钟|阅读量:

List

定义

  • List是Java中比较常用的集合类，List 是一个接口，它继承于Collection的接口。它代表着有序的队列。当我们讨论List的时候，一般都和Set作比较。

    List中元素可以重复，并且是有序的（这里的有序指的是按照放入的顺序进行存储。如按照顺序把1，2，3存入List，那么，从List中遍历出来的顺序也是1，2，3）。

    Set中的元素不可以重复，并且是无序的（从set中遍历出来的数据和放入顺序没有关系）。

  • ArrayList Vector LinkedList详细用法参考最下方的参考资料第一个链接

内部方法

  1. add(e) 添加元素
  2. addAll()
  3. get(index) 获取元素
  4. remove(index) 按索引删除元素 remove(Objiect o)按元素内容删除
  5. contains(Object o) 返回值为true/false，list中是否包含每个元素
  6. set(index,element)根据索引将元素数值改变（替换）
  7. indexOf(element) 、lastIndexOf(element),获取元素的索引

Set

特点

无序，不重复

内部方法

1. **boolean add(E e)**：添加一个元素（set中不存在该元素若）
2. **boolean addAll(Collection<? extends E> c)**：向set添加集合
3. **E ceiling(E e)**：返回大于等于给定元素的最小元素，没有返回null
4. **void clear()**：移除所有元素
5. **Object clone()**：浅拷贝集合
6. **boolean contains(Object o)**：判断set是否包含某元素
7. **E first()**：返回set的第一个元素
8. **E last()**：返回set的最后一个元素
9. **E floor(E e)**：返回给定元素的上一元素
10. **E higher(E e)**：返回比给定元素大的最小元素
11. **E lower(E e)**：返回比给定元素小的最大元素
12. **SortedSet<E> headSet(E toElement)**：返回不包含给定元素前面的所有元素
13. **SortedSet<E> tailSet(E fromElement)**：返回大于等于给定元素后面的所有元素
14. **SortedSet<E> subSet(E fromElement, E toElement)**：返回开始/结束元素之间的所有元素集合，[from, to）
15. **NavigableSet<E> headSet(E toElement, boolean inclusive)**：返回比给定元素小的元素集合，true表示小于等于
16. **NavigableSet<E> tailSet(E fromElement, boolean inclusive)**：返回比给定元素大的元素集合，true表示大于等于
17. **boolean isEmpty()**：判断set是否为空
18. **Iterator<E> iterator()**：返回一个升序的set迭代器
19. **E pollFirst()**：移除第一个元素，返回null如果set为空
20. **E pollLast()**：移除最后一个元素，返回null如果set为空
21. **boolean remove(Object o)**：移除指定元素
22. **int size()**：返回集合元素数目

常用实现类

  1. HashSet
    1. 底层结构：HashMap，使用其中的键来存储元素
    2. 元素不重复的标准：元素类的hashCode()和equals()
  2. LinkedHashSet
    • HashSet的子类，使用HashCode确定在集合中的位置，使用链表的方式确定位置（有序,按照输入的顺序输出）
  3. TreeSet
    1. 底层结构：TreeMap，使用其中的键来存储元素
    2. 特点：存储即排序
    3. 元素不重复的标准：比较器，0表示重复，非0表示不重复
